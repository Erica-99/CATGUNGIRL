Repo rules:
- Don't force push to dev or main (you don't have permission to anyway)
- Make a new branch from dev for each new feature/asset/mechanic/bugfix/etc.
- Clean up after yourself: Delete your branches when you've finished with them and merged them into dev.
- If you get a merge conflict, please check with whoever you're conflicting with before resolving it.
